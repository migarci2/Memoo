# Memoo Web (Next.js)

Frontend app for Memoo.

- `/`: public landing page
- `/login`: login
- `/register`: register
- `/onboarding`: onboarding
- `/team/[teamId]`: product workspace

## Run locally

From monorepo root (`platform/`):

```bash
npm run web:install
npm run web:dev
```

Open `http://localhost:3000`.
