# Memoo Video Studio

Branded Remotion workspace for Memoo.

## Commands

From the repository root:

```bash
npm run video:dev
npm run video:render
```

Or from this folder:

```bash
npm install
npm run dev
npm run render
```

## Current composition

- `MemooBrandStory`: 16:9 storyboard-driven explainer based on the current Memoo pitch

## Story structure

The current cut is organized into nine scenes:

1. Hook
2. Negation
3. Technical differentiator
4. Capture
5. Transformation to playbook
6. Reuse
7. Execution proof
8. Memory and collaboration
9. Final CTA

The project reuses the VAG Rounded type family and Memoo brand palette from `apps/web`.
Scenes that need real product footage are currently represented with branded UI placeholders so they can be replaced later with captures from the live app.
